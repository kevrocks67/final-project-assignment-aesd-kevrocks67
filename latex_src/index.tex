The Door Security Daemon is an embedded C application designed for the Beagle\+Bone Black (BBB). It handles pin input from an adafruit 3x4 keypad and a magnetic contact switch. The "{}lock"{} is a SG90 servo. This daemon supports secure, signed OTA updates via SWUpdate.
