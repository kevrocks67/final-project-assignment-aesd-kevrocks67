\doxysubsection{Detailed Description}
Main entry point for the Door Security Daemon. 

This module implements the top-\/level epoll event loop that orchestrates inputs from the keypad, door sensor, and system timers. It translates hardware interrupts and signals into \doxylink{structFSM}{FSM} events to control the door lock.
