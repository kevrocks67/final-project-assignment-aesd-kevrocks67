\doxysubsection{Detailed Description}
Utilities for discovering and identifying Linux input devices. 


\begin{DoxyItemize}
\item This module parses a path, /proc/bus/input/devices by default, to dynamically map device names (like "{}\+Keypad"{} or "{}\+GPIO"{}) to their corresponding /dev/input/eventX nodes. 
\end{DoxyItemize}
