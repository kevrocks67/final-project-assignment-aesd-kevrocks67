\doxysubsection{Detailed Description}
Hardware abstraction layer for servo-\/controlled door lock. 


\begin{DoxyItemize}
\item Provides an interface to control a servo motor via Linux PWM sysfs, mapping angular positions to PWM duty cycles. 
\end{DoxyItemize}
