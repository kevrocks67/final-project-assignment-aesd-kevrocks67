Implementation details for the keypad translation logic. 

\begin{DoxyNote}{Note}
This implementation specifically filters for EV\+\_\+\+KEY and value 1. Release events (value 0) are explicitly ignored to prevent double-\/triggering on mechanical key bounces. 
\end{DoxyNote}
