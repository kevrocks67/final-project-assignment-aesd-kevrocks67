\doxysubsection{Detailed Description}
Interface for the magnetic contact switch attached to the door. 


\begin{DoxyItemize}
\item This module interprets raw Linux input events from the gpio-\/keys driver and maintains the logical state (Open/\+Closed) of the door. 
\end{DoxyItemize}
