Detailed Description

Main entry point for the Door Security Daemon.

This module implements the top-level epoll event loop that orchestrates inputs from the keypad, door sensor, and system timers. It translates hardware interrupts and signals into FSM events to control the door lock.

Application entry point and event orchestration loop.

  • Initializes hardware drivers (PWM, GPIO), sets up the FSM, and enters a non-blocking epoll_wait loop to process:
  • Timer expirations (Auto-lock)
  • GPIO edge triggers (Door sensor)
  • Keypad input events (PIN entry)
  • Linux signals (Graceful shutdown)
  • Returns
    int EXIT_SUCCESS on graceful shutdown, EXIT_FAILURE on init error.
