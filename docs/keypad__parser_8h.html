Detailed Description

Public API for translating Linux input events to system commands.

This header defines the KeyEvent structure and the parsing function used by the door security system to interpret keypad input from the Linux input subsystem.

KeyEvent process_keypad_event ( const struct input_event *  ev)

Translates a Linux input_event into a system-specific KeyEvent.

  • Filters events based on type (EV_KEY) and value (1 for press). Events that do not match these criteria, or keys not defined in the switch table, return a KeyEvent with cmd set to KEY_CMD_NONE.
  • Parameters
    evPointer to the raw input_event struct read from the device.
    Returns
    KeyEvent The translated command and associated data.
